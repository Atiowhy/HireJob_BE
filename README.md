## Installation

Install Peworld API with npm :

-   Clone the Repo

```
  git clone https://github.com/ariffebri22/HirejobBE.git
```

-   Go To Folder Repo

```
  cd HirejobBE
```

-   Install Module

```
  npm install
```

-   Setup .env

```
    DB_HOST =
    DB_NAME =
    DB_USER =
    DB_PASSWORD =
    DB_PORT =
    JWT_TOKEN=
    CD_NAME =
    CD_API_KEY =
    CD_API_SECRET =
    EMAIL_NAME=
    EMAIL_PASS=
    BASE_URL=
```

-   Type `npm run start` To Start Production
